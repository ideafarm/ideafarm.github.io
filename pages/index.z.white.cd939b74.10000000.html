Step 2: Install GitHub Desktop.

Install GitHub Desktop from https://desktop.github.com/ .. Download the installation file and open it. You do not need to log in or to create an account to clone the repository or to build or install IPDOS (tm). But to obtain support or report defects, you will need to have a GitHub account. So when it prompts you to log in or create an account, you should do so. When prompted, you should enter your true name and true email address.
