Download.

Clone repository ideafarm/ideafarm.home.1 to X:\ideafarm.home.1, where X is the letter of the device that you chose. When it says, "Let's get started!", click the button that says, "Clone a repository from the Internet...". When it says, "Clone a repository", click the "URL" tab. For the URL field, carefully type "ideafarm/ideafarm.home.1". For the local path, carefully type "c:\ideafarm.home.1". Now click the "Clone" button. When it finishes, click the "History" tab to see recent updates to the repository on GitHub. To close GitHub Desktop, click the X at top right. Use Windows File Explorer to see that you now have a folder named "ideafarm.home.1" in the root folder of the device that you had selected.
