To build and install IPDOS (tm), complete the following steps:

  • Step 1: Choose a local device. Decide which local device (e.g. "C:") you want to use to store the clone of ideafarm/ideafarm.home.1 ..

  • Step 2: Install GitHub Desktop from https://desktop.github.com/ . Install GitHub Desktop.

  • Step 3: Download the files. Clone repository ideafarm/ideafarm.home.1 to X:\ideafarm.home.1, where X is the letter of the device that you chose.

  • Step 4: Edit a system setting. Add a particular folder to your computer's PATH environment variable.

  • Step 5: Build. Build IPDOS (tm).

  • Step 6: Install. Install IPDOS (tm).

  • Step 7: Edit the service settings. Edit the service settings to tell Windows that you do not want it to launch IPDOS (tm) automatically.

  • Step 8: Close. Close the service and watch all of its processes go away.

  • Step 9: report any problems. Please report any problems or defects by opening an issue on GitHub.
