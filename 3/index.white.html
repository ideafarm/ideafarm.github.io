IdeaFarm (tm) Piggyback Distributed Operating System for MS Windows

IPDOS (tm) is a general purpose software development and deployment system for the C++ programming language. It is "proprietary open source", which means that you may only use it in the ways permitted by its license. To use it, you must install it on a computer that runs the MS Windows operating system. IPDOS (tm) is distributed only in source code form and only with this GitHub repository.

WARNING: Do not install IPDOS (tm) from any other repository and do not ever use executable files obtained in any other way.

To build and install IPDOS (tm), complete the following steps:

  • Step 1: Decide which local device (e.g. "C:") you want to use to store the clone of ideafarm/ideafarm.home.1 .
  • Step 2: Install GitHub Desktop from https://desktop.github.com/ .
  • Step 3: Clone repository ideafarm/ideafarm.home.1 to X:\ideafarm.home.1, where X is the letter of the device that you chose.
  • Step 4: Add a particular folder to your computer's PATH environment variable
  • Step 5: Build IPDOS (tm)
  • Step 6: Install IPDOS (tm)
  • Step 7: Edit settings to tell Windows that you do not want it to launch IPDOS (tm) automatically
  • Step 8: Close the service
  • Step 9: Please report any problems or defects by opening an issue on GitHub
